# Semantic UI

## 使用

### 安装

1. 安装node.js

2. 安装gulp

3.  ```
   npm install semantic-ui --save
   cd semantic/
   gulp build
    ```

   ​



### 主题

1. 找到文件修改路径 `site/globals/site.variables`

2. 修改颜色文件

   ```less
   @primaryColor   : @pink;
   @secondaryColor : @grey;
   @red            : #B03060;
   @orange         : #FE9A76;
   @yellow         : #FFD700;
   @olive          : #32CD32;
   @green          : #016936;
   @teal           : #008080;
   @blue           : #0E6EB8;
   @violet         : #EE82EE;
   @purple         : #B413EC;
   @pink           : #FF1493;
   @brown          : #A52A2A;
   @grey           : #A0A0A0;
   @black          : #000000;
   ```
